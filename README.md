# Inventory Management System

一個基於 Vue 3 + Vite 開發的現代化庫存管理系統，提供直觀易用的界面來管理商品庫存。目前只為靜態展示。

## 技術棧

- **前端框架**: Vue 3
- **構建工具**: Vite
- **開發語言**: JavaScript
- **UI 組件**: Element Plus
- **路由管理**: Vue Router
